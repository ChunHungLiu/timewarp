Timewarp

SASS + ANGULAR boilerplate for single page applications to speed up your frontend development

Download .zip Download .tar.gz View on GitHub

SCSS folder

_core.scss

Is the place where to put variable declarations (colors, spacings, widths) and typography stuff (fonts imports, font variables, etc.). You will also find a ready to use set of useful mixins.

_layout.wathever.scss

The css architecture is splitted into common breakpoints to help you in responsive development. You will find a _layout.common.scss that includes common styles for all screen sizes then there files for common breakpoints: 1240up for large screens, 1030up for regular desktop screens, 768up for tablets and small laptops, 481up for phablets.
